Quick Finder is a QGIS plugin offering a single place to perform searches on project layers and online remote services.

All searches are configured in the settings panel.

Define searches for project layers

To search in layers, QuickFinder is indexing the layers in a SQLite database: the QuckFinder file. In the project search tab, create a new file using the create button. This file is related to the current project and should be kept along.

Then to add a layer search, click on . In this dialog, you will define:

  • the search name (it will be used to group the results by search name)
  • the layer
  • the field used for this search (or an expression to combine several)
  • priority

If checking record entry, the search will be directly indexed on the QuickFinder file. Uncheck this if you want to create several searches on very large layers (from ~100'000 elements). You will be able to index them all at once later.

Since the searches are indexed on an external file, editing in QGIS the layers will make the indexed file obsolete. You can define a delay after which the plugin will propose you to update the searches.

Online remote services

At the moment, you can use an Open Street Map and/or a GeoMapFish service.

Searching

A text box is displayed in QGIS toolbars. Use this to search whithin your defined project searches and remote services. All results will be displayed under it.
